# odin-recipes

**Introduction**

This project demonstrates the use of basic html foundations to build a simple recipe page. The skills used are of the following:

- HTML Boilerplate
- Headings to name the recipe and give short descriptions
- Anchor element to wrap text to the recipe page
- Use of unordered list to list ingredients
- Use of ordered list to describe step by step directions
- Image element to display an image of the recipe
